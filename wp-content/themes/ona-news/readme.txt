== Description ==

Ona News is a block WordPress theme for full site editing. It's a child theme for a popular Ona theme.
This theme is a perfect fit for a modern magazines, news, publishers, blogs, or newspaper websites.
Ona News theme provides various block patterns with post grids, like masonry, list etc. Additionally Ona offers multiple color schemes that you can edit via global styles. Customize header, footer or post layout using advanced drag and drop WordPress site editor, no page builders required. Improve your Google rankings thanks to the dynamic code generation and on-demand block editor assets.
